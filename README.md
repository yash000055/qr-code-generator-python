# QR CODE GENERATOR USING PYTHON :
##  Description
This project generates QR codes using Python.  
It demonstrates two approaches:
1. **Normal Method** – a quick and simple way to generate a QR code.
2. **Advanced Method** – allows customization such as QR version, size, border, and error correction.

The project converts text or URLs into scannable QR code images.

## Features :
- Generate QR codes from text or URLs
- Simple one-line QR code generation
- Advanced customization options
- Beginner-friendly Python project
- Clean and easy-to-understand code

## Technologies Used :
- Python
- `qrcode` library
- Pillow (used internally by the qrcode library)

## The generated QR code images are saved in the project directory as:
-normal.png
-qrcode.png
